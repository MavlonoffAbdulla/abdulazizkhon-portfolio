# DESIGN.md — Дизайн-скилл (фирменный стиль)

> Стиль согласован с существующим личным брендом автора (визитки: бело-синий дизайн,
> треугольный логотип "A"). Агент должен использовать ТОЛЬКО эти токены — никаких
> произвольных цветов/шрифтов вне этого списка.

## Цветовая палитра

| Токен | HEX | Использование |
|---|---|---|
| `primary` | `#1F4E9C` | Основной синий — кнопки, ссылки, акценты, логотип |
| `primary-light` | `#EAF1FB` | Светло-голубой фон карточек/секций-акцентов |
| `primary-dark` | `#153A73` | Hover-состояния кнопок, тёмный акцент |
| `ink` | `#1A1A1A` | Основной текст |
| `muted` | `#5B5B5B` | Второстепенный текст, подписи |
| `bg` | `#FFFFFF` | Фон страницы по умолчанию |
| `bg-alt` | `#F7F9FC` | Альтернативный фон секций (чередование) |
| `border` | `#E3E8F0` | Границы карточек, разделители |

### tailwind.config.js — фрагмент
```js
theme: {
  extend: {
    colors: {
      primary: { DEFAULT: '#1F4E9C', light: '#EAF1FB', dark: '#153A73' },
      ink: '#1A1A1A',
      muted: '#5B5B5B',
      bgAlt: '#F7F9FC',
      borderSoft: '#E3E8F0',
    },
  },
}
```

## Типографика
- Заголовки: жирный sans-serif (например, `Inter` или `Manrope`, вес 700–800).
- Основной текст: тот же шрифт, вес 400–500, `text-base` / `text-lg`.
- Никаких декоративных/рукописных шрифтов — стиль строгий, деловой (соответствует духу B2B-услуг).
- Масштаб заголовков: H1 `text-4xl md:text-6xl`, H2 `text-3xl md:text-4xl`, H3 `text-xl md:text-2xl`.

## Логотип и графический мотив
- Треугольная "A" — используется в Header (компактно) и как фоновый геометрический
  акцент в Hero (полупрозрачный, не отвлекающий от текста).
- Тот же мотив можно повторить в Footer как маленький watermark.
- QR-код с визитки на сайте не нужен — это специфично для печатной визитки.

## Компоненты — визуальные правила
- **Кнопки**: `primary` фон, белый текст, `rounded-lg`, hover → `primary-dark`.
  Вторичная кнопка — контурная (`border-primary`, текст `primary`, прозрачный фон).
- **Карточки** (услуги/проекты): белый фон, `border-borderSoft`, `rounded-xl`,
  лёгкая тень при hover (`shadow-md`), без ярких градиентов.
- **Секции**: чередование фона `bg` / `bgAlt` для визуального разделения блоков
  без использования жёстких линий-разделителей.
- **Отступы**: секции — `py-16 md:py-24`; внутренний контейнер — `max-w-6xl mx-auto px-4`.

## Адаптивность
- Mobile-first: базовые классы Tailwind без префикса — под 320–480px.
- `md:` — планшет/десктоп от 768px.
- `lg:` — широкий десктоп от 1024px+.
- Хедер на мобильных сворачивается в гамбургер-меню с тем же порядком пунктов.

## Анимации (если используются)
- Только `fade-in` + лёгкий `translate-y` при появлении секции в зоне видимости.
- Длительность 300–500ms, без "прыгающих" или бросающихся в глаза эффектов.
- Кнопки — простой `transition-colors`, без scale-эффектов.
