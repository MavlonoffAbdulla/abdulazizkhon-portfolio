# DESIGN.md — Дизайн-скилл (фирменный стиль)

> Стиль согласован с личным брендом автора — **тёмная tech-визитка**: глубокий
> тёмно-синий фон, синие акценты, логотип "A" со скобками `<>` внутри, лёгкое
> свечение элементов. Агент должен использовать ТОЛЬКО эти токены — никаких
> произвольных цветов/шрифтов вне этого списка.

## Тема: тёмная (dark-first)
Сайт выполнен в тёмной теме — это основная и единственная тема. Светлой версии нет.

## Цветовая палитра

| Токен | HEX | Использование |
|---|---|---|
| `bg` | `#0A1128` | Основной фон страницы (глубокий тёмно-синий, как на визитке) |
| `bg-alt` | `#0F1A3A` | Альтернативный фон секций (чередование), фон карточек |
| `surface` | `#16244D` | Поверхности: карточки при hover, модалки, выпадающие меню |
| `primary` | `#2E7CF6` | Основной синий акцент — кнопки, ссылки, иконки |
| `primary-bright` | `#5EA2FF` | Светло-синий — hover-состояния, градиенты, свечение |
| `primary-dark` | `#1B4FA8` | Нажатые состояния кнопок |
| `ink` | `#F2F6FF` | Основной текст (почти белый с холодным оттенком) |
| `muted` | `#8FA3C8` | Второстепенный текст, подписи |
| `border` | `#233459` | Границы карточек, разделители |
| `glow` | `#3D8BFF` | Цвет свечения (box-shadow с прозрачностью) для акцентных элементов |

### tailwind.config.js — фрагмент
```js
theme: {
  extend: {
    colors: {
      bg: { DEFAULT: '#0A1128', alt: '#0F1A3A' },
      surface: '#16244D',
      primary: { DEFAULT: '#2E7CF6', bright: '#5EA2FF', dark: '#1B4FA8' },
      ink: '#F2F6FF',
      muted: '#8FA3C8',
      borderSoft: '#233459',
      glow: '#3D8BFF',
    },
    boxShadow: {
      glow: '0 0 24px rgba(61, 139, 255, 0.25)',
      glowStrong: '0 0 40px rgba(61, 139, 255, 0.4)',
    },
  },
}
```

## Логотип
Логотип — треугольная "A" с кодовыми скобками `<>` внутри (как на визитке):
- Контур треугольника с градиентом от `primary-bright` к белому (сегментированные
  грани, как на визитке — часть граней синяя, часть светлая).
- Внутри по центру — символ `<>` тонким шрифтом.
- Реализовать как SVG-компонент `Logo.jsx` (не растровая картинка) — чтобы
  масштабировался без потерь в Header, Footer и favicon.
- В Header — компактный вариант (только знак), рядом текст "A.M." цветом `ink`.

## Типографика
- Заголовки: жирный sans-serif (`Inter` или `Manrope`, вес 700–800), цвет `ink`.
- Основной текст: тот же шрифт, вес 400–500, цвет `ink`; второстепенный — `muted`.
- Акцентные слова в заголовках можно выделять цветом `primary-bright` (как на
  визитке фамилия выделена синим).
- Масштаб заголовков: H1 `text-4xl md:text-6xl`, H2 `text-3xl md:text-4xl`, H3 `text-xl md:text-2xl`.

## Графический мотив (tech-стиль визитки)
- Тонкие «печатные дорожки» (circuit lines) — как на фоне визитки: едва заметные
  линии с точками-узлами по углам секций Hero и Footer. Реализация: SVG-паттерн
  с opacity 0.06–0.12, цвет `primary-bright`. Не перегружать — это фоновая фактура.
- Свечение (glow) на ключевых элементах: CTA-кнопка Hero, активный фильтр
  портфолио, логотип при hover — `shadow-glow` из токенов.

## Компоненты — визуальные правила
- **Кнопки**: primary фон `primary`, текст белый, `rounded-lg`, hover → `primary-bright`
  + `shadow-glow`. Вторичная — контурная (`border-primary`, текст `primary-bright`,
  прозрачный фон), hover → лёгкая заливка `primary/10`.
- **Карточки** (услуги/проекты): фон `bg-alt`, граница `borderSoft`, `rounded-xl`;
  hover → фон `surface`, граница `primary/40`, лёгкий `shadow-glow`.
- **Секции**: чередование `bg` / `bg-alt`. Разделение блоков — за счёт фона,
  без жёстких линий.
- **Бейджи/теги** (стек технологий, категории): фон `primary/10`, текст
  `primary-bright`, граница `primary/20`.
- **Отступы**: секции — `py-16 md:py-24`; контейнер — `max-w-6xl mx-auto px-4`.

## Контрастность (обязательно при тёмной теме)
- Основной текст `ink` на `bg` — контраст выше 12:1, ок.
- `muted` использовать только для второстепенного текста ≥ 14px — не для мелких подписей.
- Проверять контраст интерактивных элементов — требование Lighthouse Accessibility ≥ 85
  остаётся в силе (см. WORKFLOW.md).

## Адаптивность
- Mobile-first: базовые классы Tailwind без префикса — под 320–480px.
- `md:` — планшет/десктоп от 768px.
- `lg:` — широкий десктоп от 1024px+.
- Хедер на мобильных сворачивается в гамбургер-меню с тем же порядком пунктов.

## Анимации (2D)
- `fade-in` + лёгкий `translate-y` при появлении секции в зоне видимости.
- Длительность 300–500ms.
- Кнопки — `transition-colors` + плавное появление glow, без scale-эффектов.
- Правила 3D-анимации — в `ANIMATION_3D.md`.
